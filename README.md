# Porter's Nails and Spa — Site Files

This repo contains the static files deployed to Hostinger (portersnailsandspa.com).

## Files
- `index.html` — Public-facing booking site
- `manage.html` — Owner dashboard (deployed to /manage/ on Hostinger)
- `sitemap.xml` — SEO sitemap
- `robots.txt` — Search engine crawl rules

## Branches
- `main` — Working branch (edits happen here)
- `production` — What's live on Hostinger (Hostinger deploys from this branch)

## Deployment
Hostinger auto-deploys when changes are pushed to the `production` branch.
Only push to `production` when changes are tested and ready to go live.
